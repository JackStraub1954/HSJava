The enclosed documentation is for the support of facilities offered in the Introduction to Java class. The tools package contains classes directly related to completing exercises. Additional packages are for the support of recommended projects, including:

  • Cartesian Plane
    A project for creating graphs in the Cartesian Plane. The cartesian_plane package contains the GUI implementation and sample code demonstrating its usage.
  • Dawkins Weasel
    A project for creating graphs in the Cartesian Plane. The dawkins_weasel package contains implementation notes, and a sample implementation.
  • Matrices
    A project for the development of a class to encapsulate matrices and matrix operations. The matrices package contains implementation notes, and a sample implementation.
  • Tic Tac Toe
    A project for writing a tic-tac-toe game. The tictactoe package contains the GUI implementation and sample code demonstrating its usage.
  • Tower of Hanoi
    A project for demonstrating the Tower of Hanoi puzzle. The towerofhanoi package contains the GUI implementation and sample code demonstrating its usage.
